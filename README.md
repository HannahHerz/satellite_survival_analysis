# Satellite Survival Analysis

## Structure
- Data: datasets in different stages of cleaning and preprocessing and the scripts (see details below)
- Plots: plots for descriptive analysis and possibly later for the next steps 

### Data
- 01_data_cleaned: merged UCS and CelesTrak datasets and initial datacleaning (missings and check outliers) 
- 02_data_grouped: contains (log)-transformed variables (=numerical) and grouped variables (=categorical) that are ready to use in further analysis

### Plots
- Plots for descriptive analysis: check correlations or relationships between lifetime and covariate
